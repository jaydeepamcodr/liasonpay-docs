## Getting Started

To start using the Payments API, you'll need:

1. Your API key for authentication
2. Your store ID to identify your account
3. Customer and payment details for processing transactions

Check out the [Process Payment](/api-reference/payments/process-payment) endpoint to begin accepting payments in your application.
