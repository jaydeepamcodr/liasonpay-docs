## Getting Started

To start using the Dispute API, you'll need:

1. Your API key for authentication
2. The transaction ID for the disputed payment

Check out the [Raise Dispute](/api-reference/disputes/raise-dispute) endpoint to begin implementing dispute functionality in your application.
