## Getting Started

To start using the Subscription API, you'll need:

1. Your API key for authentication
2. Your store ID to identify your account
3. Price IDs for the subscription plans you want to offer

Check out the [Create Subscription](/api-reference/subscriptions/create-subscription) endpoint to begin implementing subscription functionality in your application.
